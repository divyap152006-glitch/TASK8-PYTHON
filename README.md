# Simple Rule-Based Chatbot -- Python

This project is a **basic rule-based chatbot** built using Python. It
responds to user inputs using simple **if-elif-else conditions**, making
it perfect for beginners learning about input handling and control flow
in Python.

## Project Overview

The chatbot: - Greets the user - Responds to phrases like "hi", "how are
you", etc. - Detects keywords such as **weather**, **college**, and
**help** - Exits when the user types **"bye"**

## Technologies Used

-   Python 3
-   Input/output handling
-   If-elif-else statements
-   While loop

## How to Run

1.  Save the chatbot script as `chatbot.py`
2.  Open a terminal and run:

```{=html}
<!-- -->
```
    python chatbot.py

3.  Start chatting! Type **bye** to exit.

## Features

-   Simple text-based chatbot
-   Keyword detection
-   Friendly responses
